# Simulation Pipeline for Activity Schedule Reconstruction

This repository provides a transparent and reproducible simulation pipeline  
for activity schedule reconstruction.

## Repository Structure

- `result_tour_gen.ipynb`  
  Contains generated simulation outputs, intermediate results, and figures.

- `seoul_tour_gen.ipynb`  
  City-specific notebook used to generate the Seoul activity schedules  
  reported in the manuscript (provided for transparency).

- `sgp_tour_gen.ipynb`  
  City-specific notebook used to generate the Singapore activity schedules  
  reported in the manuscript (provided for transparency).

- `val_tour_gen.ipynb`  
  Notebook used to generate activity schedules for the controlled experiments  
  reported in the manuscript. Simulated data will be provided upon acceptance.
